# Security Policy

## Supported Versions

Currently no version is safe for use, I'll be working on this project to release version 1.0.0 to be the first supported version.

| Version | Supported |
| - | - |
| 1.0.0 | in progress |
| < 1.0.x | :x: |

## Reporting a Vulnerability

Please use [GitHub's private vulnerability reporting](https://github.com/hugoboss17/laravel-vite-mix/security/advisories/new) to report security issues. Do not open a public issue, as that exposes the vulnerability before a fix is available.

I'll acknowledge reports within a few days and aim to release a patch as soon as possible.
